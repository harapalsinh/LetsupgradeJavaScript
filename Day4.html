There are 4 ways we can access the elements in DOM
1: By Class

we can select the elements by class name : document.getElementsByClassName("NAME_OF_CLASS");

2: By ID
we can select the elements by id : document.getElementById("ID_NAME"); 

3. By Name
we can select the elements by name: document.getElementsByName("NAME");

4. By Tag name
we can select the elements by Tag name: getElementsByTagName("h1");
